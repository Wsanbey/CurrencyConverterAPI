# Currency Converter API

## Descricao
API para conversao de moedas em tempo real.

## Funcionalidades
- Conversao entre moedas
- Taxas de cambio atualizadas
- API REST simples
- Suporte a multiplos paises

## Tecnologias
- Python
- Flask/FastAPI
- API de taxas de cambio

## Uso
`ash
GET /api/convert?from=USD&to=BRL&amount=100
`

## Instalacao
`ash
git clone https://github.com/Wsanbey/CurrencyConverterAPI.git
cd CurrencyConverterAPI
pip install -r requirements.txt
python app.py
`
